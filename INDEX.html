What is QWEN?

QWEN is an open-source interpreted programming language, developed by Jirapas Jipipob. QWEN programs are run on a QWEN virtual machine.
It was designed to be used for education. The language is available on various platforms.

History

Jirapas Jipipob began the QWEN project in 2019. QWEN was originally designed for game development.
Later the project was changed to focus on education purpose.
